### Arquitectura Multicloud Evaluación 2

**Evaluación de Arquitectura MultiCloud**

Este repositorio contiene el código fuente y la orquestación mediante Docker Compose para el sistema ERP de Farmacias SIMI, estructurado en una arquitectura de dos capas (Frontend y Backend) desplegada en instancias separadas de Amazon EC2.
